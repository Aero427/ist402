Lab Options

  • Your video should be you explaining how CSS cascades and modifying a css color, padding, or other attribute to demonstrate live. Also show how classes can be added to HTML in order to apply the CSS.

Option 2

  • Research CSS Grid, an exciting “new” way to do responsive grid based layouts in CSS

  • Use https://grid.layoutit.com/ to help build the boilerplate for your resume and then make a basic resume using this grid as a template

  • Your video should be explaining CSS Grid to the best of your abilities as well as how the tool works to save a lot of time and effort

Option 3

  • Fork one of the following codepens and fill in your details for your resume:

    • https://codepen.io/knaman2609/pen/Zbyjvv

  • Your video should show you using Codepen to reflect changes to the resume but I want you to explain how Sass or Stylus is interpreting code to generate CSS in affect

Option 4

  • Use TailwindCSS to create a basic Resume from scratch (or a boilerplate) and post the site to the web either on github or codepen [requires tooling, yarn / npm which we’ll discuss in future weeks]  or using a https://play.tailwindcss.com/ link

  • Create a hello-world video about getting started with TailwindCSS so that someone else could learn how to make a resume using this technologies
