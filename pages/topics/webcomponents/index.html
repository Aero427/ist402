Slides - Web components & Package managers

Great Web components Crash Course tutorial that's 25 minutes from nothing to decent looking card elements being used twice but built once.

Web components are the future of web development. Combined with package management systems like npm, they empower teams to leverage the work of others in a sustainable, secure and accessible manner.

This week is mostly in slide form as a result of the many talks that btopro gives on these topics. This course is built out of web components and increasingly everything from signs at fast food restaurants to websites you use ever day (like YouTube) will be built out of web components.
