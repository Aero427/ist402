Option 1

  • Try out VueJS

  • Look over some examples: https://vuejs.org/v2/examples/

  • Run through the “Scrimba” interactive videos in the Introduction https://vuejs.org/v2/guide/index.html

  • Make a very basic Vue component using an aspect of your resume to make a building block

  • Write up your thought process for selecting how your component should function

  • Do your video explaining the Vue component you made

Option 2

  • Use https://open-wc.org/ to make a boilerplate project

  • Install the tooling (will need to get NodeJS first)

  • Create a hello-world app that uses packages like:

  • @lrnwebcomponents/meme-maker
  • @lrnwebcomponents/moar-sarcasm

  • Wire them into your app (https://lit-element.polymer-project.org/try/ is another good tutorial)

  • Do a video demonstrating adding the packages via NPM and adding / using the web component in your boilerplate

  • Write a blog post with all the steps involved to replicate your demo 

Option 3

  • Work through Tutorials on React, Angular

  • https://angular.io/start  

  • https://reactjs.org/tutorial/tutorial.html

  • Which made more sense to you? Which was easier to follow? Who makes their community more approachable

  • Do a video explaining a hello world codepen from either community

  • Do a blog post about your experience, which you liked more and why.
